ME315 - Manipulação de Banco de Dados

Repositório de Desafios da Disciplina

  • Desafio 1

    Foco na importação e manipulação de dados em formato *tidy* com `dplyr`. Aborda a transformação de tabelas, agregação de dados e criação de gráficos com `ggplot2` usando dados de tuberculose.

    Visualizar Download .Rmd

  • Desafio 2

    Ensina a processar grandes bases de dados que não cabem na memória. A técnica consiste em ler e processar o arquivo em partes (*chunks*), calculando estatísticas e combinando os resultados ao final.

    Visualizar Download .qmd

  • Desafio 3

    Focado em dados relacionais, ensinando a combinar múltiplas tabelas com diferentes tipos de *joins*. O objetivo é enriquecer uma base de dados principal com informações de tabelas secundárias.

    Visualizar Download .qmd

  • Desafio 4

    Aborda a importação e manipulação de dados de planilhas Excel. Ensina a selecionar planilhas, limpar nomes de colunas, verificar o formato *tidy* e criar visualizações de séries temporais.

    Visualizar Download .qmd

  • Desafio 5

    Cria uma função em R para rastrear a rota de uma aeronave, lendo dados de voos em partes (*chunks*). O resultado é uma tabela e um mapa interativo com o trajeto e estatísticas de voo.

    Visualizar Download .qmd

  • Desafio 6

    Introdução à manipulação de bancos de dados SQLite a partir do R. O laboratório ensina a conectar, listar tabelas e executar consultas SQL para contar clientes por país e identificar músicas compradas.

    Visualizar Download .qmd

  • Desafio 7

    Laboratório de introdução ao SQLite usando R como interface. Ensina a explorar tabelas, realizar consultas com `WHERE` e `INNER JOIN`, e extrair dados para análise e visualização.

    Visualizar Download .qmd

  • Desafio 8

    Processamento de grandes volumes de dados com armazenamento em um banco de dados SQLite. Utiliza a leitura em *chunks* para filtrar e inserir dados de voos de forma eficiente em tabelas.

    Visualizar Download .qmd
